- **One-command installer** — sets up Hermes + Liberty Agent + OpenRouter API key in one step

## Install

**Linux / macOS:**
```bash
curl -fsSL https://raw.githubusercontent.com/Liberty-Emporium/hermes-agent/main/scripts/install-alexander-ai.sh | bash
```

**Windows (PowerShell):**
```powershell
irm https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.ps1 | iex
```
*(Then run the Liberty Agent manually — Windows support coming soon)*
